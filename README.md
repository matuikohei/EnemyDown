# Minecraft プラグイン EnemyDown

**開発背景**
---
私は現在、未経験からのエンジニア転職に向けてプログラミングスクールでJavaを学習しています。
このプラグインは、Minecraftのプラグイン開発を通して、Webアプリケーション開発で使用する知識・技術を学習することを目的に作成しました。

**ミニゲームを実装した理由**
---
このミニゲーム「EnemyDown」を実装した目的は３つあります。<br>１つ目は、実践的なプログラミングスキルの習得のためです。Javaを使って、実際に動作するプラグインを開発する経験を積むことで、Javaの基本文法からWebアプリケーション開発で使用する知識・技術まで学ぶことができ、実務に役立つスキルを習得することができると考えました。<br>２つ目は、データベースの知識や操作スキルの習得のためです。データベースとの連携が必要となるプラグインを開発することで、MySQLやMyBatisについて学ぶことができ、データベース操作の実践的なスキルを身につけることができると考えました。<br>３つ目は、開発能力の習得のためです。マインクラフトのプラグイン開発を通して、設計・実装・テストのサイクルを経験することで、プログラムの要件定義や設計、実際の実装中に発生するエラーが発生した際の解決方法、テストの実施方法や重要性を学ぶことができ、開発能力を高めることができると考えました。

**プラグイン概要**
---
「EnemyDown」は、プレイヤーが制限時間内に敵を倒してスコアを競うミニゲームです。ゲームの難易度を、easy、normal、hardから選択することができます。「enemydown」+「難易度」をコマンド入力することでゲームが開始します。ゲーム終了後に、スコアが表示され、スコアはデータベース（MySQL）に保存されます。「enemydown list」とコマンド入力することで、スコアの履歴を確認することができます。

**主要機能**
---
* ゲーム開始時、体力や空腹ゲージを最大化
* ゲーム開始時、装備や武器を毎回同じものへ変更
* 一定のエリア内でのみ敵が発生
* 敵の種類はランダムに決定
* 敵を倒すと点数が加算
* 敵の種類によって異なる点数を設定
* 時間制限を設定
* 時間制限が来たらエリア内の敵を消滅
* ゲーム開始時と終了時に特殊効果や状態異常を無効化
* ゲーム終了時にスコアボードで合計点数を表示
* スコア、プレイヤー名、日時をデータベース（MySQL）に保存(上書きではなく、全て保存すること)

**工夫した点**
---
* 難易度変更の実装：<br>ゲームをプレイする人のスキルレベルに合わせてゲームの難易度を調整できるよう、コマンド引数を活用して難易度の変更機能を実装しました。
* 制限時間の実装：<br>ゲームプレイ中に緊張感を感じ、ゲーム終了後には達成感を感じられるように、スケジューラを使用して、制限時間を設定しました。
* 敵の種類によって異なる点数を設定：<br>ゲームを戦略的にプレイできるように、敵の種類に応じて加算されるスコアが変動するように設計し、Switch文を用いて実装しました。
* 基底クラスの作成：<br>コードの再利用性と拡張性を高めることを目的に、基底クラス（BaseCommand）を作成しました。
* MyBatisの使用：<br>スコア情報の管理を行うために、MySQLやMyBatisを使用してデータの登録や取得を行いました。
* JUnitの使用：<br>プラグインの品質向上のため、JUnitを使用して単体テストを実施しました。
